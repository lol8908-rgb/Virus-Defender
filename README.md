## 🔧 Installation & Setup

### Voraussetzungen
- Python 3.8 oder höher
- pip (Python Package Manager)

### Installation
```bash
# Projekt klonen
git clone https://github.com/lol8908-rgb/Virus-Defender.git
cd Virus-Defender

# Abhängigkeiten installieren
pip install -r requirements.txt
```

---

## 🏗️ Bauen mit Python

Das Projekt nutzt ein reines **Python-basiertes Build-System**:

```bash
# Standard Build
python build.py

# Build mit Optimierungen
python build.py --optimize

# Clean Build
python build.py --clean
```

### Build-Optionen
- `--optimize`: Erstellt eine optimierte Version
- `--clean`: Entfernt vorherige Build-Artefakte
- `--output <path>`: Bestimmt das Ausgabeverzeichnis

---

## ▶️ Verwendung

```bash
# SafeGuard starten
python -m safeguard

# Mit Scan-Verzeichnis
python -m safeguard --scan ~/Documents

# Vollständiger Help
python -m safeguard --help
```
